Introducción a CSS

¿Qué es CSS?

CSS (Hojas de Estilo en Cascada) es un lenguaje que usamos para diseñar páginas web. Sirve para cambiar colores, tamaños, bordes, fondos, y más.

Selectores en CSS

Los selectores permiten aplicar estilos a elementos HTML. Aquí algunos ejemplos:

Selector de tipo: Aplica a todas las etiquetas como p, h1, etc.
Selector de clase: Se escribe con punto (.nombre) y se puede usar en muchos elementos.
Selector de ID: Se escribe con # (#nombre) y solo se usa una vez.
